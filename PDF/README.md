# O objetivo do relatório é identificar padrões relacionados aos acidentes de trabalho no Brasil e responder a perguntas como:

- Quais setores econômicos registraram mais acidentes?
- Quais atividades apresentaram mais óbitos?
- Quais setores possuem as maiores taxas de incidência?
- Quais ocupações concentram mais acidentes e óbitos?
- Quais estados apresentam maior volume ou risco relativo?

O relatório considera dados agregados por:

- ano;
- setor econômico — CNAE;
- ocupação — CBO;
- Unidade Federativa — UF;
- quantidade de acidentes;
- quantidade de óbitos;
- Taxa CAT;
- taxa de letalidade.

Anos analisados de 2016 a 2025.
