Esta pasta contém os dados estruturados extraídos do relatório:

Acidentes do Trabalho no Brasil — 2016 a 2025

O relatório foi elaborado a partir das Comunicações de Acidente de Trabalho — CAT — registradas junto ao INSS/eSocial.

Os arquivos presentes nesta pasta representam as tabelas e rankings publicados no documento original, incluindo informações por:

- ano;
- setor econômico — CNAE;
- ocupação — CBO;
- Unidade Federativa — UF;
- quantidade de acidentes;
- quantidade de óbitos;
- Taxa CAT;
- taxa de letalidade;

A extração foi realizada utilizando:

* Ferramenta: VSCode, Python, Pandas, Pdfplumber
